# Prompt Scoring Exercise

Now it's your turn to practice the Context + Task + Output framework from the previous lesson. In this exercise, you'll write a prompt for a realistic scenario and receive AI-powered feedback on how well you applied each dimension of the C+T+O framework.

<Callout type="tip" title="Before You Begin">
Think back to the three components: **Context** (what data is available and what's been found so far), **Task** (the specific analytical method and variables), and **Output** (the format and level of detail you need). A strong prompt addresses all three clearly and specifically.
</Callout>
